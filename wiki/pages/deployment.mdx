
# Deployment Guides

How to build and deploy Fiinny.

## Mobile App

### Android
- **Build**: `flutter build appbundle`
- **Track**: Internal Testing (Play Console)

### iOS
- **Build**: `flutter build ipa`
- **Track**: TestFlight

## Web
- **Hosting**: Firebase Hosting
- **Build**: `flutter build web` or `npm run build` (Next.js)
