# Startup Prefs (`startup_prefs.dart`)

**Path:** `lib/services/startup_prefs.dart`

## 1. Overview
A local-only preference wrapper using `SharedPreferences`.

## 2. Responsibilities
- **First Run Check**: Tracks `hasSeenWelcome()` to determine if the "Welcome / Get Started" carousel should be shown.

## 3. Usage
Checked in `main.dart` or `SlashScreen` logic.
