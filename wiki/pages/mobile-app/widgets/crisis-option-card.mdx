# Crisis Option Card

A selectable option card used within the Crisis Mode configuration flow (e.g., choosing a difficulty level).

## Overview

A simple, interactive card that displays a title, subtitle, and optional icon. It supports an "active" state for selection highlighting.

## Usage

```dart
CrisisOptionCard(
  title: 'Strict Mode',
  subtitle: 'No notifications, hard limits.',
  icon: Icons.lock,
  isActive: true,
  onTap: () => selectOption(strict),
)
```

## Styling

- **Active State**:
    - Background: `Colors.teal.withOpacity(0.11)`
    - Text/Icon: Teal shades.
    - Elevation: 5
- **Inactive State**:
    - Background: Standard toggleable card color.
    - Text/Icon: Grey/Black.
    - Elevation: 3
