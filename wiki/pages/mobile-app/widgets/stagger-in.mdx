# Stagger In

A specialized animation widget for staggering list items or elements into view.

## Overview

`StaggerIn` combines a fade and slide transition to create a smooth entry effect. It is designed to work with a parent `AnimationController` and an `Interval` to sequence items (e.g., list items appearing one after another).

## Usage

```dart
StaggerIn(
  controller: _animationController,
  interval: Interval(0.1, 0.6, curve: Curves.easeOutCubic),
  from: Offset(0, 0.1), // Slide from 10% down
  child: MyListItem(),
)
```

## Implementation Details

It chains two transitions:
1.  **FadeTransition**: Opacity goes from `fadeBegin` to `1.0`.
2.  **SlideTransition**: Position goes from `from` (offset) to `Offset.zero`.

Both are driven by the same `CurvedAnimation` derived from the provided `controller` and `interval`.
