# Stock Ticker (`stock_ticker_model.dart`)

**Path:** `lib/models/stock_ticker_model.dart`

## 1. Overview
A lightweight model for live or mocked **Market Data**.

## 2. Responsibilities
- **Quote:** `symbol`, `price`, `exchange`.
- **Identity:** `sector` and `logoUrl` for rich UI display ("Reliance Industries").

## 3. Usage
- **UI:** The "Watchlist" or "Market Movers" card.
