# AI Message (`ai_message.dart`)

**Path:** `lib/models/ai_message.dart`

## 1. Overview
Represents a chat message within the "Fiinny Brain" or "Financial Assistant" interface.

## 2. Responsibilities
- **Conversation State:** Tracks `text`, `timestamp`, and `isUser` (to differentiate user queries from AI responses).
- **Status:** Tracks delivery status (`sending`, `sent`, `error`) for UI feedback.

## 3. Usage
- **UI:** The Chat interface where users ask "How much did I spend on food?".
