# Assets Screen

The portfolio management dashboard.

## Overview

Displays the user's Assets (Investments, Property, Gold, etc.) with current values and allocation breakdowns. It uses a modern, clean design with a large total value display.

## Features

- **Categorization**: Filters by asset class (Equity, MF/ETF, Gold, Crypto, etc.).
- **Breakdown**: Shows allocation percentage by category (e.g., "Equity - 45%").
- **Asset Cards**: Displays individual assets with their current value and type-specific icons (color-coded).
- **Edit/Delete**: Long-press or menu contextual actions to manage assets.

## Architecture

- **Data**: Fetches `List<AssetModel>` from `AssetService`.
- **Filtering**: Client-side filtering of the fetched list based on the selected `_segment`.
- **State**: Simple local state `_AssetsScreenState` for managing the Future and `_segment`.
