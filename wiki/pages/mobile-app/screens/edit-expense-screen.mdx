# Edit Expense Screen

The editor for existing expense records.

## Overview

Allows users to modify details of an existing `ExpenseItem`. It shares much of the UI DNA with `AddTransactionScreen` but is structured specifically for editing.

## Features

- **Date Changing**: Allows moving the transaction date.
- **Split Adjustment**: Users can add/remove friends from an existing split.
- **Bank Reference Detection**: Automatically detects bank reference numbers (UTR, Ref No) in notes and extracts them into a separate display field (`_looksLikeBankReference`).
- **Audit Trails**: Preserves `createdAt` and `createdBy` fields while updating `updatedAt`.

## Structure

1.  **Step 0: Basics**: Amount, Category, Date, Note, Counterparty.
2.  **Step 1: People**: Payer, Friends/Group involved.
3.  **Step 2: Review**: Summary before saving.
