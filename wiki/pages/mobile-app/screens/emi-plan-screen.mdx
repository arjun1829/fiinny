# EMI Plan Screen

A clean, list-based EMI simulator.

## Overview

Helps users visualize how breaking a large payment into smaller chunks affects their cash flow.

## Features

- **Plans**: Displays standard tenures (3, 6, 9, 12 months).
- **Interest Simulation**: Adds a flat ~2% fictional interest markup for realistic simulation.
- **Selection**: "Choose" button returns the selected plan (currently mock).
