# Insight Feed Screen

The "Fiinny AI" intelligent assistant interface.

## Overview

A dual-tabbed interface combining a conversational AI chat with a proactive insights feed.

## Tabs

1.  **Chat**: 
    - **Conversational AI**: Chat with `AiChatService`.
    - **Multimodal Input**: 
        - **Voice**: Speech-to-text integration (`speech_to_text`).
        - **Vision**: Camera integration to "Scan Receipts" (`image_picker`).
    - **Action Router**: Parses intent (e.g., "Add expense 500") and executes app actions directly.

2.  **Insights**:
    - **Proactive Feed**: Generated by `FiinnyBrainService`.
    - **Models**: Displays `InsightModel` cards typed as `Critical`, `Warning`, or `Positive`.

## Architecture

- **AI Service**: Routes natural language queries to the backend (Gemini via `AiChatService`).
- **State**: Manages chat history stream and local insight generation.
