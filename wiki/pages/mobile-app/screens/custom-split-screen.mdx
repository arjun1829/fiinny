# Custom Split Screen

Advanced interface for defining unequal splits.

## Overview

Allows precise control over how an expense is divided among participants.

## Features

- **Toggle Mode**: Switch between "Split Equally" (auto-calculation) and "Custom Split" (manual entry).
- **Validation**: Prevents saving if the sum of custom splits does not equal the total amount.
- **Multi-Source**: Can handle splitting a *batch* of selected expenses (summing them up first).
