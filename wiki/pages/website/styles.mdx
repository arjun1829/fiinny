
# Styling & Theming

The Website uses **Tailwind CSS** with a custom configuration to match the mobile app's brand identity.

## Colors
Defined in `tailwind.config.ts`:

- **Primary**: `colors.teal.600`
- **Secondary**: `colors.teal.100`
- **Dark Mode**: Automatically respects system preference (using `class` strategy).

## Typography
- **Font**: [Inter](https://fonts.google.com/specimen/Inter) (Google Fonts via `next/font`).

## Components
Shared UI elements in `components/ui` (Shadcn-like structure):
- `Button`: Variants (default, outline, ghost).
- `Card`: Container with consistent padding/shadow.
