
# Shared Components

Reusable UI blocks standardizing the website's look and feel.

## UI Primitives (`components/ui`)

Low-level building blocks, mostly wrapping Radix UI primitives with Tailwind styling (shadcn/ui style).

- **`Button`**: Primary, Secondary, Ghost variants.
- **`Input`**: Standard text input with focus rings.
- **`Sheet`**: Slide-over panels (used for Mobile Nav).

## Layout Components

- **`Navbar`**: Sticky header with responsive logic.
- **`Footer`**: Site-wide footer with legal links.
- **`Container`**: Centers content with responsive max-width.

## Feature Components

- **`HeroSection`**: The visual hook of the landing page, featuring the App Showcase.
- **`FeatureGrid`**: A responsive grid layout for highlighting app capabilities.
- **`ReviewCard`**: Displays user testimonials.
